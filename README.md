## Status

🚧 **Pre-alpha** — `Navigation`, `NavigationItem`, `NavigationGroup`, `BreadcrumbsBuilder` entregues em `NAV-001..004`.

## Convenções

- `declare(strict_types=1)` em todos os ficheiros PHP
- Classes `final` por default
- Resources auto-registados via `Navigation::autoRegisterFromResources(...)`
